CNIT-131A - XML JSON Assingments

City College of San Francisco, Fall 2022

This website shows the sequences of assigments in order to learn XML and JSON formats and how we can use integrated them with HTML and CSS from two perspectives of coding: Back-End and Front-End.

Topic Description
HW1 - Course Page This assignment consisted of creating a website put away on City College San Francisco's Server to manage the ten homework corresponding to CNIT-131A class. The main section comprises a table where we can link ten tasks. Its display improved using CSS attributes for the most critical tags (h1, table,p, and anchor).
HW2 - My First XML This assignment publishes an XML file; I could understand how we can make it and how it can be linked with a CSS file to see it on Browser. Also, we apply a series of attributes with CSS, mainly with Grid display because we have a matrix related to book information.
